System Architecture

System Boundary

Neraium operates strictly as an infrastructure intelligence layer. The system analyzes telemetry and produces governed analytical output, but it does not control infrastructure systems.

Read Only

Neraium ingests operational telemetry but does not write commands or modify system state.

No Control Layer

The platform does not replace or override existing control systems such as BAS or SCADA.

Operator Authority

Infrastructure operators retain full authority over system decisions and actions.
