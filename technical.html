Technical Architecture // NER-SPEC-A

Passive Sovereignty.

01 / Unidirectional Ingestion

The Neraium node utilizes a logical data diode. Data travels from the facility to our intelligence layer, but no return path exists at the protocol level. No write-access. No command injection risk.

02 / Protocol Compatibility

  • [+] BACnet/SC (Secure Connect)
  • [+] Modbus TCP / RTU
  • [+] OPC UA Unidirectional
  • [+] MQTT over Sparkplug B

Edge Computing

Telemetry is normalized at the edge. Heavy-lift deviation analysis occurs in situ, ensuring that only governed intelligence—not raw sensitive packets—crosses the institutional boundary.
