System Boundary and Demonstration Environment

Neraium operates as a read-only infrastructure intelligence layer. It ingests telemetry, performs multivariate deviation analysis, and produces reviewable outputs without executing control commands or automated remediation.

01 — System Boundary
Operational Scope
The platform receives telemetry and produces analytical outputs. It does not manage equipment, alter control states, or perform automated system intervention.

Read-Only Ingestion

Telemetry is received without issuing control signals downstream.

Multivariate Drift Logic

Correlated signal deviation is evaluated relative to established baselines.

Operator Review Output

Outputs are structured for human review and governance documentation.

02 — Demonstration Dataset
NASA C-MAPSS Benchmark
Current analytical validation references the NASA C-MAPSS turbofan degradation dataset. This benchmark environment provides multivariate run-to-failure sequences used for drift detection modeling. It does not represent validation within water infrastructure systems.
